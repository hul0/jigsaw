JIGSAW
Joint Intelligence Generator for Strategic Attack Wordlists
Est. 2026 • Rupam Ghosh

JIGSAW is a high-performance, intelligent wordlist generator designed for modern offensive security. Constructed with the robustness of Rust, it transcends simple brute-force methods by employing probabilistic models, targeted personal profiling, and advanced permutation logic to cultivate high-value password candidates.

Installation

Ensure that Rust and Cargo are installed on your system. Compilation requires the standard release profile.

git clone https://github.com/rupam-ghosh/jigsaw.git cd jigsaw cargo build --release

The resulting binary resides within target/release/jigsaw.

Operational Modes

1. Mask Attack

Generates candidates based on a specific structural pattern.

./jigsaw -m "?l?l?l?l?d?d" --output mask_list.txt

2. Markov Chain

Trains a probabilistic model on an existing wordlist (e.g., RockYou) to generate statistically probable passwords.

./jigsaw --train rockyou.txt ./jigsaw --markov --model jigsaw.model --count 1000000

3. Personal Attack

Constructs a targeted wordlist derived from a subject's personal information (names, dates, interests) using deep permutation logic.

./jigsaw --personal --profile target.json

Use --interactive to build a profile via the wizard.

RESTful API

JIGSAW exposes its engine via a high-performance HTTP server. This allows for integration into web frontends or automated pipelines.

Start Server: ./jigsaw server --port 8080

Generate Wordlist

POST /api/personal/generate

Generates a list of password candidates from a JSON profile.

Request Body

{ "first_names": ["John"], "last_names": ["Doe"], "dates": ["1990"], "numbers": ["123"] }

Check Password

POST /api/check-password

Efficiently checks if a specific password exists within the potential mutations of a profile.

Request Body

{ "profile": { ... }, "password": "John_Doe123" }

Response

{ "found": true, "position": 715, "time_taken_ms": 3 }

Memorable Generator

GET /api/memorable/generate

Produces a secure, memorable password using an Adjective-Noun-Verb pattern.

Response

{ "password": "HappyTigerJump42!" }
